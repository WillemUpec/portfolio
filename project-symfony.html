Symfony + Twig + Chart.js

💻 Projet MMI • Développement Web & Données

Description du Projet

Ce projet est une application web dynamique complète développée avec le framework PHP Symfony. Elle vise à offrir un tableau de bord d'analyse interactif, structurant et affichant des données complexes à l'aide de représentations graphiques animées.

L'architecture s'appuie sur le modèle MVC de Symfony. Les contrôleurs gèrent la logique métier et l'accès à la base de données via Doctrine ORM. L'interface utilisateur est quant à elle entièrement intégrée avec le moteur de template Twig, utilisant notamment les Twig Component Live pour des interactions dynamiques sans rechargement de page.

Visualisation Statistique

La pièce maîtresse de l'application est son module de visualisation. J'ai utilisé la bibliothèque Javascript Chart.js pour générer des diagrammes interactifs (courbes, barres, secteurs) alimentés dynamiquement par des requêtes Ajax. Les graphiques s'animent de manière fluide lors du chargement des données ou du filtrage périodique.
